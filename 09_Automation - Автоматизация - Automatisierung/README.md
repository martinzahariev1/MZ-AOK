# 09_Automation - Автоматизация - Automatisierung

## What Belongs Here

Automation scripts, prompts, import templates, generated outputs, processing notes, and repeatable workflows.

## Naming Rules

Use date, tool or workflow name, artifact type, purpose, language, and status.

## Examples

- `2026-06-26_Automation_Template_Document-import_EN_final.md`
- `2026-06-26_Automation_Output_Email-index_EN_copy.csv`
