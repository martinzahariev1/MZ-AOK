# 08_Reports - Доклади - Berichte

## What Belongs Here

Case summaries, final reports, export packets, translated summaries, and presentation-ready materials.

## Naming Rules

Use report date, audience or topic, report type, language, and status.

## Examples

- `2026-06-26_Case-Report_AOK-summary_DE_draft.pdf`
- `2026-06-26_Case-Report_Evidence-index_EN_final.xlsx`
