# Contracts - Договори - Verträge

## What Belongs Here

Contracts, amendments, signed agreements, contract drafts, and related notices.

## Naming Rules

Use the contract date, party, document type, short subject, language, and status.

## Examples

- `2026-01-15_Partner-Company_Contract_Service-agreement_DE_original.pdf`
- `2026-02-01_Employee-Name_Contract_Employment_BG_copy.pdf`
