# Employees - Служители - Mitarbeiter

## What Belongs Here

Employee records, employment documents, personnel correspondence, identity references, and role notes.

## Naming Rules

Use employee name or ID carefully and avoid unnecessary private details in filenames.

## Examples

- `2026-01-01_Employee-Name_Record_Start-date_BG_copy.pdf`
- `2026-03-15_Employee-Name_Email_Work-schedule_DE_original.eml`
