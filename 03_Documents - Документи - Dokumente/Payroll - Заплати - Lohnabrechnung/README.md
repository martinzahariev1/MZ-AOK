# Payroll - Заплати - Lohnabrechnung

## What Belongs Here

Payslips, wage calculations, salary payments, payroll corrections, and payroll summaries.

## Naming Rules

Include payroll period, employee, document type, language, and status.

## Examples

- `2026-04-30_Employee-Name_Payslip_April-2026_DE_original.pdf`
- `2026-05-31_Payroll_Report_May-2026_DE_final.xlsx`
