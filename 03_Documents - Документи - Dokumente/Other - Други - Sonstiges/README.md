# Other - Други - Sonstiges

## What Belongs Here

Documents that do not yet fit another document category. Move files out of this folder when a better category is identified.

## Naming Rules

Use the standard naming pattern and add enough detail to make later sorting easy.

## Examples

- `2026-06-26_Unknown_Document_Unsorted_MIX_copy.pdf`
- `2026-06-26_External-Source_Note_Background-info_EN_copy.md`
