# Police - Полиция - Polizei

## What Belongs Here

Police reports, complaint filings, witness notes, police correspondence, and case numbers.

## Naming Rules

Include filing date, police office, document type, short subject, language, and status.

## Examples

- `2026-06-01_Police_Report_Complaint-DE_original.pdf`
- `2026-06-05_Police_Email_Case-number_DE_original.eml`
