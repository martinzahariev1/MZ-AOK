# 01_Timeline - Хронология - Zeitlinie

## What Belongs Here

Chronological event notes, reconstructed timelines, dated summaries, and source-backed event lists.

## Naming Rules

Use dates first so files sort chronologically:

`YYYY-MM-DD_Source_Type_Short-description_Language_Status.ext`

## Examples

- `2026-06-23_AOK_Event_Contribution-notice_DE_original.pdf`
- `2026-06-26_Case-Timeline_Summary_Main_EN_draft.md`
