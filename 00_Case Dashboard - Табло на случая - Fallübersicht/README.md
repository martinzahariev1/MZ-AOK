# 00_Case Dashboard - Табло на случая - Fallübersicht

## What Belongs Here

Central CSV trackers, case dashboards, indexes, and quick-reference files used to navigate the whole case.

## Naming Rules

Use short lowercase names for trackers and the standard evidence naming pattern for exported dashboard files:

`YYYY-MM-DD_Source_Type_Short-description_Language_Status.ext`

## Examples

- `documents.csv`
- `timeline.csv`
- `2026-06-26_Case-Dashboard_Index_Current_EN_final.xlsx`
