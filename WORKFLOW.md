# Workflow

## Intake

1. Save the original file in the correct folder.
2. Rename the file using the evidence naming rules.
3. Add an entry to the relevant CSV tracker in the case dashboard.
4. Add timeline entries when the document proves an event.

## Review

1. Check the source, date, parties, amount, and language.
2. Note contradictions or missing information in `07_Analysis - Анализ - Analyse`.
3. Link the item to people, companies, claims, payments, or legal issues where relevant.

## Reporting

1. Draft summaries in `08_Reports - Доклади - Berichte`.
2. Keep supporting evidence references in the report.
3. Export final packets with stable filenames and final status.

## Archiving

Move only inactive or superseded material to `10_Archive - Архив - Archiv`. Do not archive source evidence unless it is duplicated elsewhere and clearly indexed.
